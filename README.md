# biotrace
Application permettant le suivi des installations d'un méthaniseur

## Auteurs:
Théo LE BAIL
Thomas LEBRETON
Romain PIPON

## Requirements

- python3.10
- MariaDB/MySQL database

## Installation

```bash
virtualenv .env && source .env/bin/activate && pip install -r requirements.txt
```

## Usage

```bash
pipenv run start
```
